## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/albachteng/character_builder.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Launch the app
   ```sh
   npm run start
   ```
